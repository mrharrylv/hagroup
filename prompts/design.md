PROMPT FOR AURA.BUILD

Create a premium, modern, highly professional website design for a full-service IT company offering comprehensive technology solutions.

The company provides:
	•	IT consulting
	•	IT infrastructure maintenance and support
	•	Business website design, development, hosting, and maintenance
	•	DevOps engineering and automation
	•	Cloud architecture (AWS-focused but cloud-agnostic positioning)
	•	Cloud migrations and modernization
	•	AI integration and AI-powered automation solutions
	•	Custom software development
	•	Full-stack web application development
	•	Infrastructure as Code and CI/CD implementation
	•	Security hardening and monitoring

The design must feel:
	•	Modern and premium
	•	Clean and minimal but visually impressive
	•	Trustworthy and enterprise-ready
	•	Slightly tech-forward with subtle futuristic elements
	•	Elegant animations (smooth transitions, subtle motion effects)
	•	Interactive but not overwhelming
	•	Strong typography hierarchy
	•	Professional color palette (dark blue / deep purple / charcoal / subtle gradients)
	•	Soft shadows, glassmorphism or light neumorphism accents allowed
	•	Responsive design (mobile-first)

Include Dark Mode and Light Mode toggle.

Include multi-language support structure (EN primary, ability to expand to other languages).

⸻

Required Pages

1. Home Page
	•	Strong hero section with bold headline
	•	Short powerful tagline
	•	Call-to-action buttons (Get Consultation / Contact Us)
	•	Services overview grid (icons + short descriptions)

	•	Technology stack section (AWS, DevOps, Cloud, AI, etc.)
	•	Process section (Discovery → Architecture → Build → Deploy → Maintain)
	•	Featured projects preview
	•	Testimonials section
	•	CTA banner before footer
	•	Modern footer with navigation and legal links

	•	Cookie consent banner (GDPR compliant style)

	•	Sticky navigation bar
	•	Smooth scrolling

	•	Microinteractions on buttons
	•	SEO-friendly structure
	•	Structured footer with:
	•	Services
	•	Company
	•	Legal

Visual Style Guidelines
	•	Use premium sans-serif font pairing
	•	Large hero typography
	•	Subtle animated background gradient or abstract tech shapes
	•	Clean icons (outline style)
	•	Rounded corners (medium radius)
	•	Generous spacing and white space
	•	Grid-based layout
	•	Enterprise SaaS aesthetic

Avoid:
	•	Overcrowded layout
	•	Excessive neon
	•	Overly playful startup vibe
	•	Stocky template look

The final design should feel like a high-end DevOps / Cloud architecture consultancy capable of serving enterprise clients while remaining modern and approachable.
